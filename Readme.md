# Untitled New Project in Vulkan

## Installation

**Note**: This project is only supported on Windows.

Pre-requisites:
- [Visual Studio 2022](https://visualstudio.microsoft.com/vs/)
- [Vulkan SDK](https://vulkan.lunarg.com/sdk/home)

Clone the repository using ```git clone github.com/lukehibbertgamedev/XXXXXX.git```.
Don't forget to include the submodules.

## Vulkan SDK
Ensure the Vulkan SDK is downloaded to a global space on your desktop, typically the C drive.
